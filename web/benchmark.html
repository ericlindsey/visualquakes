How fast is the Okada model on your GPU?

VisualQuakes evaluates the full Okada (1985) dislocation solution once per pixel, every frame — that is what makes dragging a slider feel instant. This page times that computation on your machine two ways: as a WebGL2 fragment shader on your GPU (how the app actually runs) and as the same math in JavaScript on one CPU core (the double-precision port used to validate the shader). Same equations, same grid — measure the difference yourself.

WebGL2 required

This browser/device does not expose WebGL2, so the GPU benchmark cannot run here. The main app has the same requirement.

Accuracy check

Speed means nothing if the answers are wrong. GPUs compute in 32-bit floats; the trusted reference (python/, NumPy) uses 64-bit. This renders the reference scenario on your GPU, reads the raw numbers back, and compares them against the double-precision Python output shipped with the site.
